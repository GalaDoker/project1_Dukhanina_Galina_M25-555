# Лабиринт сокровищ

Это текстовая приключенческая игра, где игрок перемещается по лабиринту, решает загадки и собирает предметы для победы. Игрок может использовать ключи для открытия замков и найти сокровища.

В игре вам предстоит:
- Исследовать различные комнаты лабиринта
- Собирать предметы и использовать их для решения задач
- Решать загадки и головоломки
- Избегать ловушек
- Найти сокровище и победить!

## Установка

1. Клонируйте репозиторий:
    ```bash
    git clone <URL>
    cd <папка проекта>
    ```

2. Установите зависимости:
    
    **С помощью Poetry:**
    ```bash
    poetry install
    ```
    
    **Или с помощью Make:**
    ```bash
    make install
    ```

## Запуск

Для запуска игры выполните одну из команд:

**С помощью Poetry:**
```bash
poetry run project
```

**Или с помощью Make:**
```bash
make project
```
