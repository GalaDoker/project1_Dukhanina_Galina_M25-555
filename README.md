# Лабиринт сокровищ

Это текстовая приключенческая игра, где игрок перемещается по лабиринту, решает загадки и собирает предметы для победы. Игрок может использовать ключи для открытия замков и найти сокровища.

## Установка

1. Клонируйте репозиторий:
    ```bash
    git clone <URL>
    cd <папка проекта>
    ```

2. Установите зависимости:
    ```bash
    poetry install
    ```

## Запуск

Для запуска игры выполните команду:
```bash
poetry run project
